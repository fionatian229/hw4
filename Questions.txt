1. What is an object?

In JavaScript, an object is a variable that contains many properties and functions.

2. How do you access a property of an object?
In JavaScript, a name value is called a property. The syntax for accessing an property of an object is: 

objectName.property

The expression must evaluate to a property name.

3. What is the index of the first item in an array?
In an array, the index of the first item is 0.


4. What syntax can you use to get the third item in an array?

   For example:
  var nums = [20, 25, 30, 40]
  nums[2];  //get the 3rd item from the array above

5. How do you use a variable and a loop to get every item in an array?
Use for loop function to execute. 
For example:
for ( var i=0; i<array.length; i++) {
	console.log(array[i]);
}
